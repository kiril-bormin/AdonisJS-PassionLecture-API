# AdonisJS Passion Lecture API

## Project Overview

This project is a full-stack application built with **AdonisJS** for the backend and **Vue.js** for the frontend. It allows users to manage books, lectures, and user authentication seamlessly.

---

## Technologies Used

- **Backend**: AdonisJS
- **Frontend**: Vue.js
- **Database**: MySQL
- **Build Tools**: Vite

---

## Getting Started

Follow these steps to set up the project locally:

### Frontend Setup

1. Navigate to the `frontend` directory:
   ```bash
   cd frontend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

### Backend Setup

1. Navigate to the `backend` directory:
   ```bash
   cd backend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Generate the application key:
   ```bash
   node ace generate:key
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
